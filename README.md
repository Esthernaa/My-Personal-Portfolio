# Esther Gyan-Barimah | Developer Portfolio

A responsive single-page portfolio website showcasing my projects, skills, and contact information.  
Built with *HTML, CSS, and JavaScript*, and deployed via GitHub Pages.

---

## Features
- *Single-page layout* with smooth scrolling navigation
- *Home, About, Projects, Contact* sections
- *Responsive design* using CSS Grid and Flexbox
- *Balanced profile image* with clean styling
- *Project gallery* with equal-proportion screenshots
- *Interactive contact form* with JavaScript validation
- *GitHub profile link* for code and collaboration
